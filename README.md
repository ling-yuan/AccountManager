# AccountManage
pyqt 账号密码管理软件

## 功能
界面展示已保存的账号密码

支持导入导出，可`添加/修改/删除/搜索`账号密码，可缩小至系统托盘

软件保存的数据在本地，使用sqlite数据库，不需要联网，无需担心数据泄露
